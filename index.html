VirtSH

A free and open source virtual assistant!

Download

Features:

Weather

Web Search

Instant Shells

Calculator

And more!

Installation:

Download the zip on the github repository for this project

Open the executable (MacOS)

Compile the C++ file (Linux)

Change any thing that says 'python3' to 'python' in the C++ file and then compile, because of how lazy I am (Windows)

Make sure you have python and lua installed!

We also have a docker image for this.
